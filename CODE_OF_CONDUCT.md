# Code of Conduct

This project adheres to the [CNCF Community Code of Conduct](https://github.com/cncf/foundation/blob/main/code-of-conduct.md).

By participating in this project, you are expected to uphold this code. Please report unacceptable behavior to the [CNCF Code of Conduct Committee](https://www.cncf.io/conduct/committee/) via <conduct@cncf.io>.
